Detailed Description

Defines a comprehensive database service interface for sensor monitoring and management. This service provides data access operations for all major system entities including users, roles, access privileges, sensors, measurements, and incidents. It abstracts database operations from the rest of the application and provides connection management functionality. The interface extends IBaseService to inherit common service capabilities like initialization and status reporting.
