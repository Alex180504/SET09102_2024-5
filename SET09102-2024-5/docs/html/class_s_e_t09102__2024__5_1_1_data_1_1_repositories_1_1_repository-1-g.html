Detailed Description

Generic repository implementation that provides standard data access operations for entities with integrated caching. This class implements the repository pattern to abstract data access logic and provide a consistent, high-performance API for entity operations.

Features:

  • Memory caching for read operations with 5-minute sliding expiration
  • Automatic cache invalidation on write operations
  • Performance optimizations using EF Core's AsNoTracking for read-only queries
  • Async operations with ConfigureAwait(false) for better performance in UI contexts

This base repository can be extended by specialized repositories for specific entity types when additional query capabilities are needed.
