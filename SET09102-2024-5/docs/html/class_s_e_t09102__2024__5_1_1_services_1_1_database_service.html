Detailed Description

Provides comprehensive database operations for sensor monitoring system components. This service implements IDatabaseService and handles all data access operations including connection management, user/role administration, access control, sensor data management, and incident tracking. It uses Entity Framework Core to interact with the database and includes robust error handling with detailed logging.
